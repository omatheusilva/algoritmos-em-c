# Sistema de Aprovação de Alunos

Programa desenvolvido em linguagem C para calcular a média de um aluno com base em duas notas. Caso a média seja inferior a 7,0, o aluno realiza um exame e uma nova média é calculada para determinar aprovação ou reprovação.

## Funcionalidades

- Leitura de duas notas
- Cálculo de média
- Verificação de aprovação direta
- Aplicação de exame quando necessário
- Cálculo de média final

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estruturas condicionais (if/else)
- Operações matemáticas
- Entrada de dados
- Lógica de decisão

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

Exemplo:
```text
Digite a nota 1: 6.0\
Digite a nota 2: 6.0

Media insuficiente (6.00).\
Digite a nota do exame: 5.0\
Aprovado com media final: 5.50
```
