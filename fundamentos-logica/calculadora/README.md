# Calculadora Simples com Operações Aritméticas

Programa desenvolvido em linguagem C para realizar operações aritméticas básicas entre dois números, conforme a operação escolhida pelo usuário (+, -, *, /).

## Funcionalidades

- Leitura de dois números
- Leitura da operação matemática desejada
- Realização de soma, subtração, multiplicação e divisão
- Verificação de divisão por zero
- Exibição do resultado da operação

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estrutura condicional múltipla (switch-case)
- Operações matemáticas
- Entrada e saída de dados
- Validação de dados (divisão por zero)

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```
## Exemplo de Execução

```text
+ Soma
- Subtracao
* Multiplicacao
/ Divisao

Digite a operacao desejada: /
Digite o valor 1: 10
Digite o valor 2: 2

10.00 / 2.00 = 5.00
```
