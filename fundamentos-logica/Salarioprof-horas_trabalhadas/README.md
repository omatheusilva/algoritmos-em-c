# Comparação de Salário de Professores

Programa desenvolvido em linguagem C para calcular o salário de dois professores com base na quantidade de horas-aula ministradas e no valor da hora trabalhada, identificando qual professor possui o maior salário.

## Funcionalidades

- Leitura da quantidade de horas trabalhadas de dois professores
- Leitura do valor da hora de cada professor
- Cálculo do salário total
- Comparação entre os salários
- Identificação do maior salário

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estruturas condicionais
- Operações matemáticas
- Comparação de valores
- Entrada de dados

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```
## Exemplo de Execução

```text
Horas trabalhadas do professor 1: 40
Valor da hora do professor 1: 60
Horas trabalhadas do professor 2: 30
Valor da hora do professor 2: 50

Professor 1 recebe mais: R$ 900.00 a mais
```
