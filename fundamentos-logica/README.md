# Fundamentos de Lógica em C 🧠

Esta pasta contém os algoritmos iniciais que servem como base para o domínio da sintaxe da linguagem C e do pensamento computacional.

## O que você precisa saber para entender estes programas:
* **Sintaxe Básica:** A estrutura de um programa em C (`main`, inclusão de bibliotecas como `stdio.h`).
* **Tipos Primitivos:** Diferença entre `int` (inteiros), `float/double` (reais/numeros com vírgula) e `char` (caracteres).
* **Entrada e Saída de Dados:** Como utilizar o `printf` para exibir mensagens e o `scanf` para ler dados do utilizador.
* **Operadores:** Operadores aritméticos (`+`, `-`, `*`, `/`, `%`) e a precedência matemática entre eles.
* **Condicionais Simples:** O uso de `if` e `else` para desvios básicos no fluxo do programa.
