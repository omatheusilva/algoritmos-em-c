# Intervalo entre Horários

Programa desenvolvido em linguagem C para ler dois horários no formato HH:MM, converter os valores para minutos e calcular o intervalo de tempo entre eles, informando se o intervalo é inferior, igual ou superior a 1 hora.

## Funcionalidades

- Leitura de dois horários no formato HH:MM
- Conversão de horas para minutos
- Cálculo da diferença entre os horários
- Ajuste para virada de dia
- Classificação do intervalo de tempo

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estruturas condicionais
- Conversão de unidades (horas para minutos)
- Operações matemáticas
- Entrada de dados

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o primeiro horario (HH:MM): 08:00
Digite o segundo horario (HH:MM): 08:45

Intervalo inferior a 1 hora
```
