# Maior Valor Digitado

Programa desenvolvido em linguagem C para ler diversos valores inteiros informados pelo usuário e identificar o maior valor digitado. A leitura é encerrada quando o usuário digita -1.

## Funcionalidades

- Leitura de vários valores inteiros
- Utilização de valor sentinela (-1) para encerramento
- Identificação do maior valor digitado
- Exibição do maior valor ao final

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estrutura de repetição (while)
- Valor sentinela
- Estruturas condicionais
- Comparação de valores

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite valores inteiros (-1 para encerrar):
Digite um valor: 10
Digite um valor: 25
Digite um valor: 7
Digite um valor: -1

Maior valor digitado: 25
```
