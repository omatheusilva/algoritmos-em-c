# Multiplicação com Soma Sucessiva

Programa desenvolvido em linguagem C para ler dois números inteiros positivos e calcular o produto entre eles sem utilizar o operador de multiplicação (*), utilizando apenas soma repetida.

## Funcionalidades

- Leitura de dois números inteiros positivos
- Cálculo do produto utilizando soma sucessiva
- Utilização de estrutura de repetição
- Exibição do resultado da multiplicação

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estrutura de repetição (while)
- Operações matemáticas básicas
- Lógica de multiplicação por soma sucessiva
- Variáveis e acumuladores

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o valor 1: 5
Digite o valor 2: 3

Resultado: 15
```
