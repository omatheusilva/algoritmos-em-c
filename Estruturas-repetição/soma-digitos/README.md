# Soma dos Dígitos de um Número

Programa desenvolvido em linguagem C que lê um valor inteiro informado pelo usuário, decompõe esse número em seus dígitos individuais e calcula a soma de todos eles. Por exemplo, se o usuário informar o número `432`, o programa fará o cálculo `4 + 3 + 2` e retornará `9`.

## Funcionalidades

- Leitura de um número inteiro (suporta valores positivos e negativos).
- Decomposição de números utilizando operações de divisão inteira e módulo.
- Cálculo e exibição da soma total dos dígitos.

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Funções modulares e passagem de parâmetros.
- Estruturas de repetição (`while`).
- Manipulação matemática com os operadores `/` (quociente) e `%` (resto).
- Tratamento de números negativos.

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
=== SOMA DOS DIGITOS DE UM NUMERO ===

Digite um valor inteiro: 432

A soma dos digitos do numero 432 eh: 9
```
