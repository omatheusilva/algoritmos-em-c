# Cálculo de Série Matemática

Programa desenvolvido em linguagem C que recebe dois valores inteiros inseridos pelo usuário (`k` e `n`) e realiza o cálculo computacional da seguinte série matemática: `S = k * 1 + (k+1) * 2 + (k+2) * 3 + ... + (k+n) * (n+1)`.

## Funcionalidades

- Leitura das variáveis base para o cálculo da série.
- Validação de entrada para impedir valores negativos na variável limitadora do laço.
- Cálculo iterativo traduzindo a fórmula matemática para linguagem de programação.

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Funções modulares e passagem de parâmetros.
- Tradução de lógica matemática abstrata para código computacional.
- Estruturas de repetição (`for`).
- Atualização e acúmulo de variáveis em laço.

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
=== CALCULO DE SERIE MATEMATICA ===

Digite o valor de K: 2
Digite o valor de N (nao negativo): 3

O valor final da serie S eh: 56
```
