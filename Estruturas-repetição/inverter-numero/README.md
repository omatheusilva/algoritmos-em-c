# Inversão de Dígitos de um Número

Programa desenvolvido em linguagem C que lê um valor inteiro não negativo informado pelo usuário e calcula um novo valor inteiro cuja representação decimal possui os mesmos dígitos, porém em ordem estritamente inversa. Exemplo: ao ler o valor `235`, o programa gera e exibe o valor `532`.

## Funcionalidades

- Leitura de um número inteiro.
- Validação de entrada para garantir que o número não seja negativo.
- Inversão da ordem dos dígitos matematicamente.

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Funções modulares e passagem de parâmetros.
- Inicialização correta de variáveis para evitar lixo de memória.
- Estruturas de repetição (`while`).
- Manipulação matemática com os operadores `/` (quociente) e `%` (resto) para construir novos números.

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
=== INVERSAO DE DIGITOS ===

Digite um valor inteiro nao negativo: 235

Numero original: 235
Numero invertido: 532
```
