# Divisão por Subtrações Sucessivas

Programa desenvolvido em linguagem C que lê dois valores inteiros positivos e calcula o quociente e o resto da divisão do primeiro pelo segundo. O cálculo é feito exclusivamente através de subtrações sucessivas, sem a utilização dos operadores matemáticos de divisão (`/`) e módulo (`%`).

## Funcionalidades

- Leitura de dois números inteiros positivos (dividendo e divisor)
- Validação para impedir divisão por zero
- Cálculo do quociente através de laços de repetição
- Cálculo do resto da divisão através de laços de repetição

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Funções modulares
- Estruturas de repetição (`while`)
- Estruturas condicionais (`if`)
- Lógica de programação aplicada a matemática básica

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
=== DIVISAO POR SUBTRACOES SUCESSIVAS ===
Digite o valor do dividendo (v1): 17
Digite o valor do divisor (v2): 5

Resultado da divisao de 17 por 5:
Quociente (Valor da /): 3
Resto (Valor da %): 2
```
