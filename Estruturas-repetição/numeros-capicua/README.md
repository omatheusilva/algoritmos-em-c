# Busca de Números Capicua (Palíndromos)

Programa desenvolvido em linguagem C que varre o intervalo numérico entre 10 e 9999 em busca de números "capicua". Um número capicua (ou palíndromo numérico) apresenta a peculiaridade de ter o mesmo valor quer seja lido da esquerda para a direita, quer seja lido da direita para a esquerda. Exemplos clássicos incluem `33`, `272` e `4884`.

## Funcionalidades

- Varredura automatizada em um intervalo predefinido (10 a 9999).
- Reaproveitamento de lógica para inversão da ordem dos dígitos.
- Verificação lógica por meio da comparação do número original com o seu inverso.
- Formatação da saída de dados no terminal para facilitar a leitura.

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Funções modulares e reaproveitamento de código.
- Estruturas de repetição (`for` e `while`).
- Estruturas condicionais (`if`).
- Manipulação matemática com os operadores `/` (quociente) e `%` (resto).
