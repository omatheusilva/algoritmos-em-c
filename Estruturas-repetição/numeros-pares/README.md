# Números Pares até 1000

Programa desenvolvido em linguagem C para ler um número inteiro N e exibir todos os números pares a partir de N até 1000, utilizando estrutura de repetição. Caso o número informado seja ímpar, ele é ajustado para o próximo número par.

## Funcionalidades

- Leitura de um número inteiro N
- Verificação se o número é par ou ímpar
- Ajuste automático para o próximo número par (se necessário)
- Exibição dos números pares até 1000

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estrutura de repetição (while)
- Estruturas condicionais
- Operador módulo (%)
- Sequência numérica

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o valor de N: 7

8
10
12
14
16
...
1000
```
