# Tabuada de um Número

Programa desenvolvido em linguagem C para ler um número inteiro e exibir a tabuada desse número de 1 até 10 utilizando estrutura de repetição.

## Funcionalidades

- Leitura de um número inteiro
- Geração da tabuada de 1 a 10
- Exibição formatada dos resultados

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Estrutura de repetição (while)
- Operações matemáticas
- Entrada de dados
- Saída formatada

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```
