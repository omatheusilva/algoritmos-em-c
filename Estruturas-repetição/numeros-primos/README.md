# Verificador de Números Primos

Programa desenvolvido em linguagem C que recebe um número inteiro digitado pelo usuário e analisa matematicamente se este número é classificado como "Primo". Na matemática, números primos são números naturais maiores que 1 que só podem ser divididos por 1 e por eles mesmos.

## Funcionalidades

- Leitura de um número inteiro.
- Validação automática de números negativos, 0 e 1 (que não são primos).
- Lógica de verificação otimizada: o laço de repetição é interrompido imediatamente assim que um divisor válido é encontrado.
- Exibição clara do resultado no terminal.

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Funções modulares e retorno de valores (lógica Booleana 0/1).
- Estruturas de repetição (`for`) com lógica otimizada de interrupção.
- Estruturas condicionais (`if` e `else`).
- Manipulação matemática com o operador de módulo (`%`).

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
=== VERIFICADOR DE NUMEROS PRIMOS ===

Digite um numero inteiro: 17

O numero 17 EH PRIMO.
```
