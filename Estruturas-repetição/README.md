# Estruturas de Repetição 🔁

Nesta seção, os algoritmos exploram a capacidade do computador de realizar tarefas repetitivas de forma automatizada e eficiente.

## O que você precisa saber para entender estes programas:
* **Laço `for`:** Utilizado quando sabemos previamente quantas vezes o bloco de código deve ser executado (iterações contáveis).
* **Laço `while`:** Ideal para situações onde a repetição depende de uma condição lógica que pode ser alterada durante a execução.
* **Contadores e Acumuladores:** Variáveis que controlam o número de voltas ou que somam valores progressivamente.
* **Sentinelas e Condições de Parada:** Lógicas para evitar o "loop infinito" e garantir que o programa termine no momento certo.
