# Vetor sem Valores Repetidos

Programa desenvolvido em linguagem C para realizar o preenchimento de um vetor com 500 números inteiros sem permitir valores repetidos.

## Funcionalidades

- Leitura de números inteiros
- Verificação de repetição
- Validação de entrada
- Armazenamento sem duplicidade

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas condicionais
- Estruturas de repetição
- Validação de dados

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o valor para a posicao 0: 10
Digite o valor para a posicao 1: 20
Digite o valor para a posicao 2: 10

Valor repetido! Digite outro.
```
