# Vetor Invertido em C

Programa desenvolvido em linguagem C para realizar a leitura de 8 números inteiros armazenados em um vetor e exibir os valores na ordem inversa.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Exibição invertida dos elementos

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas de repetição
- Manipulação de índices

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 10
Digite o 2º numero: 20
Digite o 3º numero: 30

Vetor invertido:
30 20 10
```
