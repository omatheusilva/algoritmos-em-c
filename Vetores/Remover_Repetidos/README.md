# Valores Únicos em Vetor

Programa desenvolvido em linguagem C para realizar a leitura de 10 números inteiros armazenados em um vetor e exibir apenas os valores únicos, ignorando repetições.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Verificação de números repetidos
- Exibição de valores únicos

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas condicionais
- Estruturas de repetição
- Busca e comparação de valores

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 5
Digite o 2º numero: 3
Digite o 3º numero: 5
Digite o 4º numero: 8

Valores unicos:
5 3 8
```
