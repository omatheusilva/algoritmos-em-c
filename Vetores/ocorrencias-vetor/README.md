# Número Mais Frequente em Vetor

Programa desenvolvido em linguagem C para realizar a leitura de 10 números inteiros armazenados em um vetor e identificar o número que aparece com maior frequência.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Contagem de frequência dos valores
- Identificação do número mais frequente

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas condicionais
- Estruturas de repetição
- Contagem de frequência
- Ponteiros

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite 10 numeros:
1 2 3 2 4 2 5 6 2 7

Numero que mais aparece: 2
Quantidade de vezes: 4
```
