# Interseção entre Vetores em C

Programa desenvolvido em linguagem C para realizar a leitura de dois vetores inteiros e identificar os elementos em comum entre eles, sem repetir valores no vetor resultado.

## Funcionalidades

- Leitura de dois vetores
- Verificação de elementos em comum
- Remoção de elementos repetidos
- Exibição do vetor de interseção

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas condicionais
- Estruturas de repetição
- Interseção de conjuntos

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Vetor A:
1 2 3 4 5

Vetor B:
3 4 5 6 7

Vetor C (interseccao sem repeticao):
3 4 5
```
