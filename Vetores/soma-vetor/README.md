# Soma de Valores em Vetor

Programa desenvolvido em linguagem C para realizar a leitura de 8 números inteiros, armazenando-os em um vetor e exibindo a soma acumulada e a soma final dos valores informados.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Cálculo de soma acumulada
- Exibição da soma final

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 5
Digite o 2º numero: 3

Soma acumulada ate o momento: 5
Soma acumulada ate o momento: 8

Soma final dos elementos: 8
```
