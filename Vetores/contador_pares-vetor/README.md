# Contador de Números Pares

Programa desenvolvido em linguagem C para realizar a leitura de 10 números inteiros armazenados em um vetor e contabilizar quantos valores são pares.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Verificação de números pares
- Contagem de valores pares

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas condicionais
- Operador módulo (`%`)

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 10
Digite o 2º numero: 7
Digite o 3º numero: 4
Digite o 4º numero: 9

Quantidade de numeros pares: 2
```
