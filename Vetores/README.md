# Vetores (Arrays Unidimensionais) 📊

Esta pasta contém exercícios focados no armazenamento de coleções de dados do mesmo tipo em posições sequenciais de memória.

## O que você precisa saber para entender estes programas:
* **Indexação:** Compreender que, em C, o primeiro elemento de um vetor está sempre na posição `0`.
* **Acesso à Memória:** Como ler e escrever em posições específicas utilizando os parênteses retos `[]`.
* **Percorrimento (Traversing):** O uso obrigatório de estruturas de repetição (geralmente `for`) para manipular cada elemento do array.
* **Limites do Vetor:** A importância de respeitar o tamanho definido na declaração para evitar erros de segmentação (*Segmentation Fault*).
* **Aritmética Modular aplicada a Índices:** Técnicas como o operador `%` para criar rotações ou deslocamentos circulares nos dados.
