# Cálculo da Média de Valores em Vetor

Programa desenvolvido em linguagem C para realizar a leitura de 6 números inteiros armazenados em um vetor e calcular a média dos valores informados.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Soma dos elementos
- Cálculo da média aritmética

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas de repetição
- Operações matemáticas

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 10
Digite o 2º numero: 20
Digite o 3º numero: 30
Digite o 4º numero: 40
Digite o 5º numero: 50
Digite o 6º numero: 60

A media dos numeros eh: 35.00
```
