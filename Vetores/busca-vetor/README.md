# Busca de Valor em Vetor

Programa desenvolvido em linguagem C para realizar a leitura de 10 números inteiros armazenados em um vetor e permitir a busca de um valor específico informado pelo usuário.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Busca de valor no vetor
- Exibição das posições encontradas

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Busca linear
- Estruturas condicionais
- Estruturas de repetição

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 5
Digite o 2º numero: 10
Digite o 3º numero: 5

Digite um valor para buscar: 5

O valor 5 esta na posicao 1
O valor 5 esta na posicao 3
```
