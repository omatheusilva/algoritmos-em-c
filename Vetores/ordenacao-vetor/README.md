# Ordenação de Vetor em Ordem Crescente

Programa desenvolvido em linguagem C para realizar a leitura de 10 números inteiros armazenados em um vetor e exibir os valores ordenados em ordem crescente.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Ordenação crescente dos elementos
- Exibição do vetor ordenado

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas de repetição
- Algoritmos de ordenação
- Troca de valores

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 8
Digite o 2º numero: 3
Digite o 3º numero: 10
Digite o 4º numero: 1

Vetor ordenado em ordem crescente:
1 3 8 10
```
