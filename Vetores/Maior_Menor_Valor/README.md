# Maior e Menor Valor em Vetor

Programa desenvolvido em linguagem C para realizar a leitura de 10 números inteiros armazenados em um vetor, identificando o maior e o menor valor informado pelo usuário.

## Funcionalidades

- Leitura de números inteiros
- Armazenamento em vetor
- Identificação do maior valor
- Identificação do menor valor

## Tecnologias Utilizadas

- Linguagem C
- Biblioteca `stdio.h`

## Conceitos Aplicados

- Vetores
- Funções
- Estruturas de repetição
- Estruturas condicionais

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
Digite o 1º numero: 10
Digite o 2º numero: 5
Digite o 3º numero: 25

Menor valor encontrado: 5
Maior valor encontrado: 25
```
