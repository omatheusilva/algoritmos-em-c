# Somatório de Matriz 3x3

Programa introdutório sobre manipulação de matrizes (arrays bidimensionais) em C. O algoritmo solicita ao usuário o preenchimento de uma matriz 3x3 e, em seguida, calcula e exibe a soma de todos os elementos digitados.

## Funcionalidades

- Leitura formatada de dados bidimensionais, indicando as posições exatas (linha e coluna) para o usuário.
- Uso de laços de repetição aninhados para percorrer a estrutura de dados.
- Função modular dedicada ao cálculo da soma total.

## Conceitos Aplicados

- Constantes (`#define`).
- **Matrizes (Arrays Bidimensionais)**.
- Passagem de matrizes como parâmetro em funções.
- Laços `for` aninhados.

## Como Compilar

Utilizando GCC:

```bash
gcc main.c -o programa
```

## Como Executar

Linux/macOS:

```bash
./programa
```

Windows:

```bash
programa.exe
```

## Exemplo de Execução

```text
=== SOMATORIO DE MATRIZ 3x3 ===

Digite os 9 valores para preencher a matriz:
Elemento [0][0]: 1
Elemento [0][1]: 2
Elemento [0][2]: 3
Elemento [1][0]: 4
Elemento [1][1]: 5
Elemento [1][2]: 6
Elemento [2][0]: 7
Elemento [2][1]: 8
Elemento [2][2]: 9

O valor do somatorio de todos os elementos e: 45
```
